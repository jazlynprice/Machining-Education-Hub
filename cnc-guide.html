What you will learn (explained)

  • Origins and work offsets (G54…G59), tool length vs. work offset
  • Workholding options: vise, clamps, soft jaws; when to use each
  • Tool selection and chip load basics with a calculator
  • Dry run, single block, and feed hold techniques

Project walkthrough

  1. Model a simple 50×50×6 mm plate in CAD
  2. In CAM, define stock, set WCS, and generate a 2D pocket
  3. Simulate, post G‑code, set work offset, then dry‑run before cutting
