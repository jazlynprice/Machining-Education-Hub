Sketching that drives models

Constrain geometry fully. Dimensions should express design intent — for example, set a slot width to match fastener size rather than arbitrary numbers.

CAM setups and heights

Stock and zero

Match stock size and choose a zero you can probe repeatably. Keep clearance heights safe but not wastefully high.

Tools and holders

Pick realistic stick‑out and holder geometry in CAM so collision checks mean something.

Simulation and prove‑out

Simulate for collisions and gouges. In the shop, run high‑Z air cuts, then step down carefully. Log the final feeds and speeds that worked so the next run is faster.
